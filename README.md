Este proyecto es una herramienta robusta desarrollada en C para el monitoreo y la predicción realista de niveles de contaminación. El sistema permite registrar datos históricos hora tras hora y utiliza algoritmos para proyectar la calidad del aire futura basándose en las tendencias acumuladas.

--Características--

Registro Horario: 
Permite la entrada de datos de contaminación en tiempo real.

Predicciones Realistas: 
Algoritmo ajustado para calcular tendencias basadas en el histórico registrado.

Eficiencia en C: 
Ejecución rápida y bajo consumo de recursos.


Para que el programa funcione correctamente, es necesario compilar todos los módulos fuente. Sigue estos pasos en tu terminal:

clona el repositorio:

git clone https://github.com/tu-usuario/pollution-predictor.git
cd pollution-predictor

despues compila los archivos, con el siguiente comando:

gcc main.c funciones.c prediccion.c -o pollution_app

y despues ejecuta el pollution_app que se produzca a partir de ahi,
Se han compilado archivos desde el 2006, lo que permite una experiencia mas realista.
